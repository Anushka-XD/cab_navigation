# Uber Fare Macro

Purpose:
- Open Uber
- Set destination
- Read fare
- Save fare to /tmp/uber_fare.txt

Steps:
1. Launch Uber app
2. Tap destination field
3. Type destination (from last input)
4. Select first suggestion
5. Wait for fare to load
6. Read fare text
7. Write value to /tmp/uber_fare.txt
8. Exit app
