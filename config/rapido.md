# Rapido App Guide

## Navigation Overview
- Specialized for quick rides (Bike and Auto-rickshaw)
- Simple two-option interface: Bike or Auto (Shuttle)
- Large destination search bar
- Fare displayed clearly after destination is set

## Key UI Elements
- **Destination Search**: Top search bar
- **Ride Type Tabs**: Two options - "Bike" and "Auto"
- **Fare Card**: Shows price, ETA, distance
- **Book Button**: Large "Request Ride" button at bottom
- **Account**: Profile icon (top-right)

## How to Get Fare
1. Allow location access
2. Tap search bar and enter destination
3. View fare for currently selected ride type
4. Tap on "Bike" or "Auto" tab to compare
5. Fares update automatically with address change

## Ride Types
- **Bike (Rapido Bike)**: Two-wheeler, faster
- **Auto (Shuttle/Auto-rickshaw)**: Three-wheeler, economical

## Safety Features
- Share ride details with emergency contacts
- Live ride tracking
- Driver name, photo, and rating displayed
- "Report" and "Help" options available

## Tips
- Bike rides faster but higher traffic risk
- Auto rides cheaper and three-wheeler advantage
- Some areas may not have bike availability
- Location accuracy affects fare accuracy
- Ride cancellation is free up to driver acceptance
