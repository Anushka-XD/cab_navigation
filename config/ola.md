# Ola App Guide

## Navigation Overview
- Home screen has "Where to?" search bar centrally placed
- "Book Now" tab for immediate rides
- Ride categories shown in vertical scrollable list
- Promotional offers/discounts displayed prominently

## Key UI Elements
- **Search Bar**: Center of screen - tap to enter destination
- **Ride Categories**: Vertical list (Ola Prime, Ola Auto, Ola Bike, etc.)
- **Fare Display**: Shows below each category with â‚¹XXX and time
- **Confirm Ride**: Large blue button to initiate booking
- **Menu**: Account icon (top-right)

## How to Get Fare
1. Confirm current location (icon bottom-left)
2. Tap "Where to?" and enter destination
3. Scroll through ride categories to see fares
4. Each category shows: Fare amount and pickup time

## Booking Steps
1. Select your preferred ride category
2. Review the fare and details
3. Tap on the category card to book
4. Confirm pickup location
5. Book confirmation shows driver assignment

## Ride Categories
- **Ola Prime**: Standard sedan
- **Ola Auto**: Auto-rickshaw (three-wheeler)
- **Ola Bike**: Two-wheeler
- **Ola Plus**: Premium/larger vehicle

## Special Features
- Offers/Coupons visible before booking
- Emergency contact sharing (SOS)
- Pre-scheduled rides available
- Share your ride with others

## Tips
- Location must be enabled for accurate fares
- "No drivers available" message means category unavailable
- Apply promo codes before confirming booking
