# Uber App Guide

## Navigation Overview
- Main screen shows search bar at top for entering destination
- Ride type selector horizontally scrollable (UberGo, UberX, etc.)
- Fare estimate shown after destination is set
- Pickup and dropoff locations can be swapped with arrow icon

## Key UI Elements
- **Search Bar**: Top of screen - tap to enter destination
- **Ride Types**: Horizontal scroll carousel below search bar
- **Fare Display**: Shows estimated price and time prominently
- **Request Button**: Large button at bottom to request ride
- **Menu**: Hamburger icon (top-left) for profile and settings

## How to Get Fare
1. Open app and confirm location
2. Tap on "Where to?" search bar
3. Enter destination address or POI name
4. Swipe through ride types to see all options and prices

## Booking Steps
1. Select desired ride type from carousel
2. Review fare and driver details
3. Tap "Confirm" button
4. App shows driver assignment and arrival time

## Common Issues
- If location isn't detecting, toggle location permission
- If fares don't load, refresh by pulling down
- Invalid addresses won't show fare - try generic location names

## Tips
- Scheduled rides appear in separate tab
- Rating and driver info appears after confirmation
- Cancel anytime before driver accepts
- Share ride details via "Share trip" feature
